# Анализ эффективности email-рассылок

## Описание
Анализ ключевых метрик email-маркетинга и исследование факторов, влияющих на эффективность рассылок (временные ряды, темы писем).

## Задачи
1. Оценить эффективность email-рассылок по ключевым метрикам:
   - **DR** (Delivery Rate) - доставляемость;
   - **OR** (Open Rate) - открываемость;
   - **CTOR** (Click to Open Rate) - кликабельность среди открывших;
   - **UR** (Unsubscribe Rate) - отписки.
2. Проанализировать динамику метрик во времени.
3. Выявить наиболее привлекательные темы писем.
4. Исследовать связь между открываемостью и отписками.

## Инструменты
- **Python** (pandas, numpy, matplotlib/seaborn)
- **Jupyter Notebook**
- **DataLens** (Динамика эффективнности рассылок и конверсий: https://datalens.yandex/81jp9ixhjs1ws)

## Файлы
- `Email_Campaign_Effectiveness.ipynb` - Jupyter ноутбук с полным анализом
- `Email_dynamics_and_conversions.png` - скриншот дашборда с метриками

## Данные
- Источник: Google Sheets.
- Период: апрель 2021 - май 2022.
- Объем: 218 записей (рассылок).
- Поля: дата, тема письма, сегмент, количество отправленных/доставленных/открытых/кликов/отписок, воронка продаж (3 шага).

## Запуск проекта
1. Откройте Email_Campaign_Effectiveness.ipynb в Jupyter Notebook / Google Colab.
2. Запустите все ячейки последовательно.
3. Данные загружаются автоматически из Google Sheets.
